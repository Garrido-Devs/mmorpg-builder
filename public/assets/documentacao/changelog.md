# Changelog

Todas as mudanças notáveis do projeto serão documentadas aqui.

---

## [0.3.0] - 2024-01-13

### Adicionado
- Sistema de autenticação completo (login, registro, logout)
- Sistema de times com convites por link
- Sistema de projetos colaborativos
- Colaboração em tempo real via Pusher
- Dashboard do usuário
- Página de times
- Página de projetos
- Blog/Features

### Componentes MMORPG
- Resource Node (mineração, pesca, lenhador, ervas)
- Crafting Station (forja, fogão, bancada, tear)
- Bank (sistema de armazenamento)
- Skill (sistema de habilidades)
- Equipment (sistema de equipamentos)
- Prayer Altar (altares de oração)
- Agility Obstacle (obstáculos de agilidade)
- Farming Patch (canteiros de agricultura)

### Melhorias
- TransformEditor agora usa eventos ao invés de polling
- ComponentEditor com modais para arrays e objetos
- Memoização de componentes React
- CSS responsivo
- Error boundaries
