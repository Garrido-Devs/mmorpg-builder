# Primeiros Passos com MMORPG Builder

Bem-vindo ao MMORPG Builder! Este guia vai te ajudar a criar seu primeiro projeto.

## Criando um Projeto

1. Acesse o **Dashboard** apos fazer login
2. Clique em **Novo Projeto**
3. De um nome para seu jogo
4. Selecione o time (ou crie um novo)

## Interface do Editor

O editor e dividido em areas:

- **Viewport**: Area principal onde voce visualiza e edita a cena 3D
- **Inspetor**: Painel direito para editar propriedades do objeto selecionado
- **Assets**: Biblioteca de modelos 3D disponiveis
- **Hierarquia**: Lista de todos os objetos na cena

## Controles Basicos

| Acao | Controle |
|------|----------|
| Mover camera | Botao direito + arrastar |
| Zoom | Scroll do mouse |
| Selecionar | Clique esquerdo |
| Mover objeto | W |
| Rotacionar | E |
| Escalar | R |

## Adicionando Objetos

1. Abra a aba **Assets**
2. Escolha uma categoria (Personagens, Ambiente, Props)
3. Clique no asset desejado
4. O objeto sera adicionado na cena

## Salvando seu Projeto

- **Salvar Local**: Baixa um arquivo JSON do mapa
- **Auto-save**: Salva automaticamente na nuvem a cada 30 segundos
