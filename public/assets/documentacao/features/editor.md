# Editor de Mapas

O Editor de Mapas é a ferramenta principal para criar os cenários do seu jogo MMORPG.

## Visão Geral

O editor permite criar, posicionar e configurar todos os elementos do seu jogo de forma visual e intuitiva.

## Interface

### Viewport (Centro)
- Visualização 3D do mapa
- Navegação com OrbitControls
- Seleção de objetos por clique

### Painel de Hierarquia (Esquerda)
- Lista de todos os objetos na cena
- Organização hierárquica
- Busca e filtros

### Painel de Inspeção (Direita)
- Propriedades do objeto selecionado
- Transform (posição, rotação, escala)
- Componentes anexados
- Ações (adicionar componente, deletar)

### Biblioteca de Assets (Inferior)
- Modelos 3D disponíveis
- Drag & drop para adicionar

## Controles

### Navegação
- **Botão esquerdo + arrastar**: Rotacionar câmera
- **Botão direito + arrastar**: Pan
- **Scroll**: Zoom

### Transformação
- **G**: Modo de translação
- **R**: Modo de rotação
- **S**: Modo de escala
- **Escape**: Cancelar seleção
- **Delete**: Remover objeto

### Atalhos
- **Ctrl+S**: Salvar (em breve)
- **Ctrl+Z**: Desfazer (em breve)
- **Ctrl+Y**: Refazer (em breve)

## Adicionando Objetos

1. Abra a Biblioteca de Assets
2. Encontre o modelo desejado
3. Clique para entrar no modo de posicionamento
4. Clique no mapa para posicionar
5. Ajuste a posição com o TransformControls

## Configurando Componentes

1. Selecione um objeto
2. No painel de inspeção, clique em "Adicionar Componente"
3. Escolha o tipo de componente
4. Configure as propriedades

## Dicas

- Use snap para alinhar objetos
- Agrupe objetos relacionados
- Nomeie os objetos de forma descritiva
- Salve frequentemente

## Colaboração

Quando trabalhando em equipe:
- Você verá os cursores de outros usuários
- Mudanças são sincronizadas automaticamente
- Conflitos são resolvidos automaticamente
