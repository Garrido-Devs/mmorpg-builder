# Trabalhando em Equipe

O MMORPG Builder permite que múltiplas pessoas trabalhem no mesmo projeto simultaneamente, com sincronização em tempo real.

## Criando um Time

1. Acesse o Dashboard
2. Clique em "Novo Time"
3. Dê um nome e descrição ao time
4. Pronto! Você é o dono do time.

## Convidando Membros

### Via Link de Convite
1. Acesse as configurações do time
2. Copie o link de convite
3. Compartilhe com quem deseja convidar
4. A pessoa acessa o link e entra automaticamente

### Gerando Novo Link
- Links podem ser regenerados
- Links antigos são invalidados ao gerar novos
- Útil para controlar quem tem acesso

## Papéis no Time

### Owner (Dono)
- Controle total do time
- Pode deletar o time
- Pode promover/rebaixar membros
- Pode regenerar links de convite

### Admin (Administrador)
- Pode convidar membros
- Pode remover membros (exceto owner)
- Pode criar/editar projetos
- Pode gerenciar permissões

### Member (Membro)
- Pode acessar projetos
- Pode editar conteúdo
- Não pode gerenciar membros

## Colaboração em Tempo Real

### Como Funciona
Quando você e outros membros estão no mesmo projeto:
- Mudanças são sincronizadas automaticamente
- Você vê os cursores de outros usuários
- Seleções são mostradas em cores diferentes

### Indicadores Visuais
- **Avatares**: Mostra quem está online
- **Cursores coloridos**: Posição de cada usuário
- **Status de sync**: Indica se está sincronizado

### Resolução de Conflitos
Quando duas pessoas editam o mesmo objeto:
- Sistema de versionamento evita perda de dados
- Última edição prevalece
- Notificação em caso de conflito

## Boas Práticas

### Comunicação
- Combine quem trabalha em qual área
- Use chat externo para coordenar
- Avise antes de mudanças grandes

### Organização
- Divida o trabalho por área do mapa
- Um edita NPCs, outro edita terreno
- Evite editar o mesmo objeto simultaneamente

### Salvamento
- Mudanças são salvas automaticamente
- Indicador mostra status de sincronização
- Sempre verifique se está "Sincronizado"

## Problemas Comuns

### Mudanças Não Aparecem
1. Verifique sua conexão
2. Recarregue a página
3. Verifique o status de sync

### Conflito de Edição
1. Aguarde o outro usuário terminar
2. Recarregue para ver a versão mais recente
3. Refaça suas mudanças se necessário

### Usuário Não Aparece Online
1. A sessão expira após 5 minutos de inatividade
2. Peça para a pessoa recarregar
3. Verifique se estão no mesmo projeto

## Limitações

- Máximo de 10 usuários simultâneos por projeto
- Sincronização pode ter delay de até 2 segundos
- Arquivos muito grandes podem demorar mais
